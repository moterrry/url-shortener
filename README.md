# url-shortener
Node.JS simple URL shortener

I wrote this for fun. Probably not ready for production use, although it seems fine for now.

# Information
- It uses SQLite
- Very simple to setup
- It has an analytics graph to see how many clicks were done (there's no verification so as long as someone goes to it it's gonna update)
- Simple UI

I wouldn't use this for production but it's definitely usable as a base.
